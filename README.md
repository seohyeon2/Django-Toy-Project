### costory_complete
* http://127.0.0.1:8000/posts
* 나만의 글을 작성할 수 있는 일기 사이트
* 새로운 글 등록, 글 수정, 글 삭제 가능
* 글 저장 시, 저장한 시간이 기록되며 수정 시, 수정일을 명시 함
* 게시글은 한 페이지당 최대 6개의 글이 보이며, 인덱스로 이동 가능
* 화면 왼쪽 상단의 로고를 클릭할 시, 메인 화면으로 이동됨
* 겹치는 제목, 10글자 이하의 짧은 내용, 지정된 특수문자 외 특수문자 사용 시 경고와 함께 글 등록이 안되도록 함
